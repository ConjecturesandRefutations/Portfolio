    A couple of freelance projects, some projects from Ironhack bootcamp and a couple of projects just for fun.

  • PLUG

    A full-blown social media website. These are just a few of the features:
    1. Create posts and blogs.
    2. Add/remove/accept/decline/request friends
    3. Comment on posts and add feedback to blogs.
    4. Like or dislike blogs and posts (think Youtube likes/dislikes).
    5. Receive Notifications.
    6. Upload a profile image.
    7. Send and receive direct messages.

  • X-Racing

    A super fun racing game. Built with Canvas, 'Vanilla' JavaScript, PHP and a MySQL database. The game includes:
    1. A choice of 3 car colors.
    2. A canvas background that continuously loops downwards.
    3. A difficulty that increases every 20 seconds.
    4. A bonus box that adds 50 to your score.
    5. The ability to play on mobile using 'TouchEvents', as well as on PC.
    6. The ability to login, which automatically saves your score to the scoretable.

  • Blog Spot

    A full-stack application built with PHP and a MySQL Database. Styled with Materialize. In a sense, this is like Twitter, but instead of being limited to 120 characters, you can write entire essays.
    1. Post, edit and delete your own blogs (with full formatting options).
    2. If you're not ready to publish your blog, you can save it as a draft.
    3. View your and others' blogs.
    4. Create, update and delete feedback on others' blogs.
    5. Like or dislike blogs (think Youtube likes/dislikes).
    6. Upload a profile image.
    7. Send and receive direct messages.

  • Four Elms (FREELANCE)

    My client required a home-guide for renting out his house. I created him this fully responsive React application styled with Tailwind CSS. A couple of notable features: the tenant can send feedback to the landlord, and the user can toggle between English and Spanish.

  • Skin Deep

  • Kick-Off!

    A football-themed Video Game built using Vanilla JavaScript. Making extensive use of Canvas Animation. Can be played on mobile. Customisable settings.

  • Drinks Rater

  • Spaced

    A very beautiful and fun space-themed shooting/collision-avoider game. It has some cool features, such as the in-game music controls and the different options of canvas background and spacecraft. The controls work on both mobile and desktop.

  • Snake!

    My rendition of the classic Nokia game built using Vanilla JavaScript. Making extensive use of Canvas Animation. Features three difficulty settings. Can be played on mobile.
